What is Style Adjuster?

Style Adjuster is a utility for tweaking CSS style definitions and seeing changes immediately in the web page. The underlying Javascript framework allows for easy definition of HTML-based editors to edit different types of CSS values.

Style Adjuster acts on CSS property definitions in CSS rules that exist in the web page. It is intended for use where you have already defined properties in CSS rules that you know need to be included in your design, but you want to efficiently try different values for those properties and make a subjective judgement about them when seeing the changes made directly in the web page.
